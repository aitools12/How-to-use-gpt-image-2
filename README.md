## What is GPT Image 2 Dev?

GPT Image 2 Dev is a web app for generating and editing images from text prompts and uploaded references. It is positioned around practical image creation tasks such as marketing creatives, product shots, social assets, and remix-style edits.

## Why use GPT Image 2 Dev?

It gives users a fast browser workflow for visual ideation without installing design software. The product is useful when the goal is to move quickly from prompt to usable image variations, then refine those results with lightweight editing.

## How GPT Image 2 Dev works

1. Open the official site and start a new image workflow.
2. Enter a prompt or upload a reference image to guide generation or editing.
3. Review outputs, iterate with revised prompts, and export the selected result.
